# Yukiko

Rewrite of the node-toukufm-slackbot in a cleaner and more consistent way.
Based on [easy-slackbot](https://www.npmjs.com/package/easy-slackbot).

## Installation

* Close this repo
* Add your own API key in `index.js`
* `$ npm install`
* `$ node index`
